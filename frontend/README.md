# 🚀 ConvoX – AI Powered Real-Time Chat Application

ConvoX is a modern real-time chat application built with the **MERN Stack**, **Socket.IO**, and **Google Gemini AI**. It provides instant messaging, media sharing, and AI-powered features in a clean and responsive interface.

---

## ✨ Features

### 🔐 Authentication
- Secure User Signup & Login
- JWT Authentication
- Password Encryption
- Logout Support

### 💬 Real-Time Messaging
- Instant One-to-One Chat
- Online / Offline Status
- Typing Indicator
- Auto Scroll
- Delivered Status

### 📎 Media Sharing
- Send Images
- Send Voice Messages

### 🤖 AI Features
- AI Chat Assistant (Google Gemini)
- AI Chat Summary
- AI Message Translation

### 📝 Message Features
- Reply to Messages
- Edit Messages
- Delete Messages
- Search Messages

### 🎨 User Experience
- Responsive UI
- Smooth Animations
- Modern Chat Interface
- Emoji Support

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/taniyarana01/ConvoX.git
```

## Backend

```bash
cd backend
npm install
npm run dev
```

## Frontend

```bash
cd frontend
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000

MONGODB_URI=your_mongodb_connection

JWT_SECRET=your_secret_key

CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret

GEMINI_API_KEY=your_gemini_api_key
```
