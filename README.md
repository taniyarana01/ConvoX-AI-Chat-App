<p align="center">

A modern **AI-powered Real-Time Chat Application** built using the **MERN Stack**, **Socket.IO**, and **Google Gemini AI**.

Real-time messaging • AI Assistant • AI Chat Summary • AI Translation • Voice Messages • Image Sharing

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/taniyarana01/ConvoX-AI-Chat-App.git
```

## Backend

```bash
cd backend
npm install
npm run dev
```

## Frontend

```bash
cd frontend
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000

MONGODB_URI=your_mongodb_uri

JWT_SECRET=your_jwt_secret

CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret

GEMINI_API_KEY=your_gemini_api_key
```
